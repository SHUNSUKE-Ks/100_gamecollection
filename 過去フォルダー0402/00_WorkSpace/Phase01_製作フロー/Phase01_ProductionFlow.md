# Phase 01 製作フロー詳細

このドキュメントでは、Phase 1: 企画・設計段階での具体的な手順を定義します。

## 1. コンセプト定義 (GamePlanner)
- [ ] ゲームジャンルの確定
- [ ] ターゲット層の選定
- [ ] コア・メカニクスの定義（どのような体験をユーザーに提供するか）

## 2. 世界観・キャラクター設計 (GamePlanner / ScenarioWriter)
- [ ] 舞台設定（時代、場所、社会情勢など）
- [ ] 主要キャラクターの設定（名前、外見、性格、目的）
- [ ] 相関図の作成

## 3. シナリオプロット作成 (ScenarioWriter)
- [ ] ストーリーの起承転結（または序破急）の構成
- [ ] 各エピソードの要約
- [ ] プロットの承認（GamePlannerによる確認）

## 4. データ構造・システム設計 (GamePlanner / Programmer)
- [ ] シナリオ用JSONのスキーマ定義
- [ ] キャラクター、アイテム、フラグなどのID命名規則の策定
- [ ] 拡張性（将来的な機能追加）の考慮

## 5. 製作基準の策定 (All Departments)
- [ ] 画像規格（解像度、アスペクト比）の決定
- [ ] 音声規格（フォーマット、サンプリングレート）の決定
- [ ] フォルダ構造とファイル命名規則の最終確認

---
> [!IMPORTANT]
> Phase 1の成果物は、以降のすべての開発工程の「正解」となります。
> 矛盾がないか、実現可能かをこの段階で徹底的に検証してください。
