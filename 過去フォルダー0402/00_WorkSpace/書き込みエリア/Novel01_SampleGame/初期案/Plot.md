# シナリオプロット: Novel01_SampleGame

## 全体構成
1.  **序章:** 塔への挑戦とエリスとの出会い。
2.  **第1章:** 下層エリアの探索。簡単な謎解きと戦闘（選択肢）。
3.  **第2章:** 中層エリア。エリスの記憶のフラッシュバック。
4.  **クライマックス:** 最上階への到達。塔の正体とエリスの決断。
5.  **エンディング:** 選択による複数結末（再会エンド / 別離エンド / 真エンド）。

## エピソード詳細

### Episode 01: 序章「塔の麓にて」
**タイトル:** 始まりの扉
**登場人物:** アルト、エリス
**舞台:** 塔の入り口（石造りの巨大な扉前）
**あらすじ:**
冒険者アルトは、伝説の「アイオーンの塔」の前に立つ。
扉が開かず困っていると、どこからともなく少女エリスが現れる。
彼女は「鍵」となる言葉を呟き、扉を開く。彼女は何者なのか？

**主要イベント:**
- [ ] アルトの独白（塔への憧れ）
- [ ] エリス登場
- [ ] 選択肢: エリスにどう声をかけるか？
    - A: 「君は誰だ？」 (警戒フラグ+1)
    - B: 「ありがとう、助かったよ」 (好感度+1)
- [ ] 塔の中へ進む

**必要な素材タグ:**
- 背景: `bg_tower_gate_day` (塔の入り口・昼)
- 立ち絵: `chara_eris_normal`, `chara_eris_smile`
- BGM: `bgm_mystery_intro`

### Episode 02: 第1章「試練の回廊」
**タイトル:** 魔力の残滓
**登場人物:** アルト、エリス、(魔物)
**舞台:** 塔内部（1階層・青白い光の回廊）
**あらすじ:**
内部は外見よりも遥かに広い空間が広がっていた。
最初の試練として、魔力で動く石像（ガーディアン）が立ちはだかる。

**主要イベント:**
- [ ] 塔内部の描写
- [ ] ガーディアン遭遇
- [ ] 選択肢: どう切り抜ける？
    - A: 正面から戦う (HP減少、勇気+1)
    - B: 弱点を探す (知力+1、エリスの助言あり)
- [ ] 戦闘勝利（テキスト描写のみ or 簡易演出）

**必要な素材タグ:**
- 背景: `bg_tower_floor01`
- 立ち絵: `chara_eris_worry`
- SE: `se_battle_start`, `se_magic_hit`
