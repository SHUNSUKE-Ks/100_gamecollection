## 全体構成

```
┌─────────────────────────────────────────────────┐
│ collection-screen (100vw × 100vh)               │
├─────────────────────────────────────────────────┤
│ nav-primary                                     │
│ [アイテム][装備][スキル]...[ライブラリー]...[戻る] │
├─────────────────────────────────────────────────┤
│ nav-secondary (library選択時のみ)               │
│ (地名)(キャラ)(NPC)(エネミー)(アイテム)...      │
├─────────────────────────────────────────────────┤
│ collection-content (flex: 1)                    │
│ ┌─────────────────────────────────────────────┐ │
│ │ [タイトル]            [ViewSwitcher 📋🖼️📊]│ │
│ ├─────────────────────────────────────────────┤ │
│ │                                             │ │
│ │   TableView / GalleryView / DetailView      │ │
│ │                                             │ │

└─────────────────────────────────────────────────┘
```
