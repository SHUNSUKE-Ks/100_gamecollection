# 命名規則

グラフィック素材のファイル名ルールです。

---

## 📝 基本フォーマット

```
{カテゴリ}_{名前}_{バリエーション}.{拡張子}
```

**ルール:**
- 全て **半角英数字** + アンダースコア
- **小文字のみ** 使用
- スペース、日本語禁止

---

## 🏷️ カテゴリプレフィックス

| プレフィックス | カテゴリ | 例 |
|---------------|---------|-----|
| `ch_` | キャラクター立ち絵 | `ch_hero_normal.png` |
| `face_` | 顔グラフィック | `face_hero_smile.png` |
| `bg_` | 背景 | `bg_forest_morning.jpg` |
| `cg_` | イベントCG | `cg_ep01_ending.png` |
| `icon_` | アイコン | `icon_potion.png` |
| `ui_` | UI素材 | `ui_button_cancel.png` |
| `fx_` | エフェクト | `fx_magic_fire.png` |
| `enemy_` | 敵グラフィック | `enemy_slime.png` |

---

## 👤 キャラクター素材

```
ch_{キャラ名}_{表情/ポーズ}.png
```

### キャラ名例
| キャラ | 名前 |
|-------|------|
| 主人公 | `hero` |
| ヒロイン | `heroine` |
| 謎の魔法使い | `mage` |

### 表情/ポーズ例
| 表情 | コード |
|------|--------|
| 通常 | `normal` |
| 笑顔 | `smile` |
| 怒り | `angry` |
| 悲しみ | `sad` |
| 驚き | `surprised` |
| 照れ | `shy` |

### 完成例
```
ch_hero_normal.png
ch_hero_smile.png
ch_heroine_angry.png
ch_mage_mysterious.png
```

---

## 🖼️ 背景素材

```
bg_{場所}_{時間/状態}.jpg
```

### 場所例
| 場所 | コード |
|------|--------|
| 森 | `forest` |
| 街 | `town` |
| 城 | `castle` |
| ダンジョン | `dungeon` |
| 塔 | `tower` |

### 時間/状態例
| 状態 | コード |
|------|--------|
| 朝 | `morning` |
| 昼 | `noon` |
| 夕方 | `evening` |
| 夜 | `night` |
| 雨 | `rain` |

### 完成例
```
bg_forest_morning.jpg
bg_town_night.jpg
bg_castle_evening.jpg
```

---

## 🎒 アイコン素材

```
icon_{カテゴリ}_{名前}.png
```

### カテゴリ例
| カテゴリ | コード |
|---------|--------|
| ポーション | `potion` |
| 武器 | `weapon` |
| 防具 | `armor` |
| 素材 | `material` |
| 貴重品 | `key` |

### 完成例
```
icon_potion_hp.png
icon_weapon_sword.png
icon_material_slime.png
```

---

## ⚠️ 禁止事項

- ❌ 日本語: `キャラ_勇者.png`
- ❌ スペース: `ch hero normal.png`
- ❌ 大文字混在: `ch_Hero_Normal.png`
- ❌ ハイフン: `ch-hero-normal.png`
