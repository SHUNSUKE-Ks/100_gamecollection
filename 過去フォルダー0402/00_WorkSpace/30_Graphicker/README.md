# 🎨 Graphicker (グラフィッカー) フォルダ

グラフィック素材の制作・納品に関する資料をまとめたフォルダです。

---

## 📁 フォルダ構成

```
30_Graphicker/
├── README.md                  (このファイル)
├── 01_素材規格書.md            (サイズ・形式ルール)
├── 02_命名規則.md              (ファイル名ルール)
├── 03_納品フォルダ構成.md       (保存先ルール)
├── 発注リスト/                 (制作依頼一覧)
└── 納品/                      (完成素材配置)
```

---

## ✅ 作業開始前に確認すること

| 確認項目 | 取得先 | 状態 |
|---------|-------|------|
| キャラクター設定書 | GamePlanner | [ ] |
| 画像サイズ規格 | 本フォルダ `01_素材規格書.md` | [ ] |
| 命名規則 | 本フォルダ `02_命名規則.md` | [ ] |
| 必要素材リスト | ScenarioWriter or GamePlanner | [ ] |

---

## 📐 サイズ規格 (クイックリファレンス)

| 素材タイプ | 推奨サイズ | 形式 |
|-----------|-----------|------|
| 立ち絵 (全身) | 1024 x 2048 px | PNG (透過) |
| 立ち絵 (バストアップ) | 512 x 512 px | PNG (透過) |
| 背景 | 1920 x 1080 px | JPG/PNG |
| アイコン | 64 x 64 px | PNG (透過) |
| UI素材 | 可変 | PNG (透過) |

---

## 📝 命名規則 (クイックリファレンス)

```
{カテゴリ}_{名前}_{バリエーション}.{拡張子}
```

### 例
| 素材 | ファイル名 |
|------|-----------|
| 主人公・通常 | `ch_hero_normal.png` |
| 主人公・笑顔 | `ch_hero_smile.png` |
| 森の背景・朝 | `bg_forest_morning.jpg` |
| ポーション | `icon_potion_hp.png` |

## 🔗 他部署との連携

### 必要な情報 (受け取る)

| 情報 | 提供元 | 用途 |
|------|-------|------|
| キャラ設定 (外見、性格) | GamePlanner | デザインの参考 |
| 必要な表情・ポーズ一覧 | ScenarioWriter | 制作リスト作成 |
| 背景リスト | ScenarioWriter | 制作リスト作成 |

### 提供する情報 (渡す)

| 情報 | 提供先 | 用途 |
|------|-------|------|
| 完成素材 | Programmer | ゲームへの組み込み |
| 素材タグ一覧 | ScenarioWriter | シナリオでの指定用 |

---

## 📦 納品方法

1. 完成した素材を `納品/` フォルダに配置
2. Asset Manager (`tools/asset-manager`) でリネーム・振分け
3. または直接 `src/assets/` へ規則に従って配置
