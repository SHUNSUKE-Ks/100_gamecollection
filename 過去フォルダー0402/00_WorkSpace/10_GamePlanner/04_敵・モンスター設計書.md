# 敵・モンスター設計書

ゲーム内の敵・モンスターのカラム定義です。

---

## 👾 エネミーカラム一覧

| カラム名 | 型 | 必須 | 説明 | 例 |
|----------|-----|------|------|-----|
| `id` | string | ✅ | 一意のID | `"monster_slime_001"` |
| `name` | string | ✅ | 表示名 | `"スライム"` |
| `label` | string | ✅ | カテゴリラベル | `"SLIME"`, `"BEAST"` |
| `image` | string | ✅ | 画像パス | `"enemy/slime.png"` |
| `rarity` | number | - | レアリティ (1-5) | `1` |
| `level` | number | ✅ | 基本レベル | `3` |
| `tags` | string[] | - | 検索用タグ | `["WEAK", "WATER"]` |

### ステータス (stats)

| カラム名 | 型 | 必須 | 説明 | 例 |
|----------|-----|------|------|-----|
| `stats.hp` | number | ✅ | 最大HP | `100` |
| `stats.mp` | number | - | 最大MP | `20` |
| `stats.atk` | number | ✅ | 攻撃力 | `15` |
| `stats.def` | number | ✅ | 防御力 | `8` |
| `stats.matk` | number | - | 魔法攻撃力 | `5` |
| `stats.mdef` | number | - | 魔法防御力 | `5` |
| `stats.spd` | number | - | 素早さ | `10` |
| `stats.luk` | number | - | 運 | `5` |

### その他

| カラム名 | 型 | 必須 | 説明 | 例 |
|----------|-----|------|------|-----|
| `skills` | string[] | - | 使用スキル名 | `["溶解液", "体当たり"]` |
| `resistance` | object | - | 属性耐性 (%) | `{"fire": -50, "water": 50}` |
| `immunities` | string[] | - | 状態異常無効 | `["poison"]` |
| `ai` | string | - | 行動AI | `"aggressive"`, `"defensive"` |
| `exp` | number | - | 獲得経験値 | `50` |
| `gold` | number | - | 獲得ゴールド | `30` |
| `drops` | object[] | - | ドロップアイテム | 下記参照 |
| `description` | string | - | 説明テキスト | `"初心者向けのモンスター"` |

---

## 🏷️ カテゴリラベル定義

| ID | ラベル | 特徴 |
|----|--------|------|
| `SLIME` | 粘体 | 物理耐性高め、魔法弱い |
| `BEAST` | 魔獣 | バランス型、素早い |
| `UNDEAD` | アンデッド | HP高い、光属性弱点 |
| `DEMON` | 悪魔 | 魔法攻撃型、闇属性 |
| `DRAGON` | ドラゴン | 全ステ高め、ボス級 |
| `HUMAN` | 人型 | バランス型 |
| `MACHINE` | 機械 | 物理耐性、雷弱点 |
| `PLANT` | 植物 | 回復持ち、炎弱点 |
| `INSECT` | 昆虫 | 素早い、防御低い |
| `ELEMENTAL` | 精霊 | 属性特化型 |

---

## 🎲 ドロップ定義

```json
"drops": [
    {
        "itemId": "material_slime_001",
        "chance": 50,
        "minCount": 1,
        "maxCount": 3
    },
    {
        "itemId": "potion_hp_001",
        "chance": 10,
        "minCount": 1,
        "maxCount": 1
    }
]
```

| カラム名 | 型 | 必須 | 説明 |
|----------|-----|------|------|
| `itemId` | string | ✅ | ドロップアイテムID |
| `chance` | number | ✅ | ドロップ率 (%) |
| `minCount` | number | - | 最小個数 |
| `maxCount` | number | - | 最大個数 |

---

## 🤖 AI定義

| ID | 説明 |
|----|------|
| `random` | ランダムに行動 |
| `aggressive` | 攻撃優先 |
| `defensive` | 回復・防御優先 |
| `support` | バフ・デバフ優先 |
| `boss` | ボス専用 (フェーズ切り替え) |
