# 武器・防具設計書

ゲーム内で使用する装備品のカラム定義です。

---

## ⚔️ 武器カラム一覧

| カラム名 | 型 | 必須 | 説明 | 例 |
|----------|-----|------|------|-----|
| `id` | string | ✅ | 一意のID | `"sword_iron_001"` |
| `name` | string | ✅ | 表示名 | `"鉄の剣"` |
| `type` | string | ✅ | 武器タイプ | `"sword"`, `"staff"`, `"bow"` |
| `icon` | string | - | アイコン画像パス | `"weapon/sword.png"` |
| `rarity` | number | - | レアリティ (1-5) | `2` |
| `price` | number | - | 購入価格 | `500` |
| `atk` | number | ✅ | 攻撃力補正 | `15` |
| `matk` | number | - | 魔法攻撃力補正 | `0` |
| `critRate` | number | - | クリティカル率 (%) | `5` |
| `hitRate` | number | - | 命中率補正 (%) | `0` |
| `element` | string | - | 属性 | `"fire"`, `"ice"`, `"none"` |
| `skills` | string[] | - | 付与スキルID | `["skill_fire_001"]` |
| `requiredLevel` | number | - | 装備可能レベル | `10` |
| `requiredClass` | string[] | - | 装備可能職業 | `["warrior", "knight"]` |
| `description` | string | - | 説明文 | `"量産品の鉄剣。"` |

---

## 🛡️ 防具カラム一覧

| カラム名 | 型 | 必須 | 説明 | 例 |
|----------|-----|------|------|-----|
| `id` | string | ✅ | 一意のID | `"armor_leather_001"` |
| `name` | string | ✅ | 表示名 | `"レザーアーマー"` |
| `slot` | string | ✅ | 装備部位 | `"body"`, `"head"`, `"accessory"` |
| `icon` | string | - | アイコン画像パス | `"armor/leather.png"` |
| `rarity` | number | - | レアリティ (1-5) | `1` |
| `price` | number | - | 購入価格 | `300` |
| `def` | number | ✅ | 防御力補正 | `10` |
| `mdef` | number | - | 魔法防御力補正 | `5` |
| `hp` | number | - | HP補正 | `0` |
| `mp` | number | - | MP補正 | `0` |
| `evasion` | number | - | 回避率補正 (%) | `0` |
| `resistance` | object | - | 属性耐性 | `{"fire": 10, "ice": -10}` |
| `requiredLevel` | number | - | 装備可能レベル | `5` |
| `description` | string | - | 説明文 | `"丈夫な革製の鎧。"` |

---

## 🏷️ 武器タイプ定義

| ID | ラベル | 特徴 |
|----|--------|------|
| `sword` | 剣 | バランス型、物理攻撃 |
| `greatsword` | 大剣 | 高威力・低速 |
| `dagger` | 短剣 | 低威力・高速・高クリ |
| `staff` | 杖 | 魔法攻撃向け |
| `bow` | 弓 | 遠距離攻撃 |
| `spear` | 槍 | リーチ長め |
| `axe` | 斧 | 高威力 |
| `fist` | 格闘 | 連続攻撃向け |

---

## 🗂️ 装備部位定義

| ID | ラベル |
|----|--------|
| `weapon` | 武器 |
| `head` | 頭 |
| `body` | 胴体 |
| `arms` | 腕 |
| `legs` | 脚 |
| `accessory` | アクセサリ |

    "atk": 25,
    "matk": 5,
    "critRate": 5,
