# 🎮 GamePlanner フォルダ

ゲームプランナー向け、ゲーム素材（アイテム・武器・スキル・アビリティ等）の設計資料をまとめたフォルダです。

## 📁 フォルダ構成

```
10_GamePlanner/
├── README.md                    (このファイル)
├── 01_アイテム設計書.md          (アイテムカラム定義)
├── 02_武器・防具設計書.md        (装備カラム定義)
├── 03_スキル・アビリティ設計書.md  (スキルカラム定義)
├── 04_敵・モンスター設計書.md     (エネミーカラム定義)
├── templates/                   (JSON/CSVテンプレート)
└── 納品/                        (完成データ配置用)
```

## 🚀 作業の流れ

1. 各設計書を読んでカラム定義を理解
2. `templates/` のテンプレートをコピーして素材を設計
3. 完成したら `納品/` に配置してエンジニアに報告
