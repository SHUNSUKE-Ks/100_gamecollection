# 音声規格書

BGM・効果音の形式・品質に関する規格です。

---

## 🎵 ファイル形式

| 形式 | 用途 | 備考 |
|------|------|------|
| **MP3** | 全般 | 互換性最優先 |
| **OGG** | 全般 | 圧縮効率良好 |
| WAV | 元データ保管用 | 納品には不使用 |

---

## 📊 BGM規格

| 項目 | 値 |
|------|-----|
| 形式 | MP3 または OGG |
| サンプルレート | 44100 Hz |
| ビットレート | 192 kbps (CBR推奨) |
| チャンネル | ステレオ (2ch) |
| ループ | シームレスループ対応推奨 |

### ループポイントについて

- 可能であれば **ループ開始/終了ポイント** をメモで添付
- または **イントロ無し版** を別途用意

---

## 🔊 SE (効果音) 規格

| 項目 | 値 |
|------|-----|
| 形式 | MP3 または OGG |
| サンプルレート | 44100 Hz |
| ビットレート | 128 kbps |
| チャンネル | モノラル (1ch) または ステレオ |
| 長さ | 通常 0.5〜3秒程度 |

### SE注意事項

- **無音部分を除去** (余計な余白を入れない)
- **音量正規化** (突出した音量差を避ける)

---

## 🗣️ ボイス規格

| 項目 | 値 |
|------|-----|
| 形式 | MP3 または OGG |
| サンプルレート | 44100 Hz |
| ビットレート | 128 kbps |
| チャンネル | モノラル (1ch) |

---

## 📏 音量レベル目安

| 素材タイプ | 目安 (dBFS) |
|-----------|------------|
| BGM | -14 〜 -10 |
| SE | -12 〜 -6 |
| ボイス | -12 〜 -6 |

---

## ⚠️ 注意事項

1. **ノイズ除去**: ホワイトノイズ、クリッピング注意
2. **フェード**: 必要に応じてフェードイン/アウト
3. **命名**: 必ず `02_命名規則.md` に従う
