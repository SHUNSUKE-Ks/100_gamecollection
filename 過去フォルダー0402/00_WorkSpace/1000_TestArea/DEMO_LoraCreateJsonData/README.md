# Tag Image Manager Demo

## 📍 概要
画像アセットのタグ管理を効率化するためのモダンなWeb UIツールです。

## 🚀 起動方法

このデモは、ブラウザのセキュリティ制限（CORS）のため、簡易Webサーバー経由で起動する必要があります。

### Pythonを使用する場合 (推奨)
1. ターミナルを開き、このフォルダに移動します。
2. 以下のコマンドを実行します。
   ```bash
   python -m http.server 8000
   ```
3. ブラウザで `http://localhost:8000` を開きます。

### Node.jsを使用する場合
Node.jsがインストールされている場合、以下のコマンドで起動できます（`-y` をつけるとインストール確認をスキップできます）。
```bash
npx -y serve
```
表示されるアドレス（通常は `http://localhost:3000`）にアクセスしてください。

### VSCodeを使用する場合
`Live Server` 拡張機能がインストールされている場合は、`index.html` を右クリックして 「Open with Live Server」 を選択してください。

## ✨ 主な機能
- **Gallery View**: 画像カード形式で一覧表示。ホバーアニメーション付き。
- **Table View**: 詳細情報をリスト形式で表示。
- **Interactive Tagging**: 各画像の下にある入力欄でタグを即座に追加（Enterキー）。
- **Fast Filter**: 上部の検索バーからファイル名やタグで高速フィルタリング。
- **Modern UI**: グラスモーフィズムとダークモードを採用したプレミアムなデザイン。
