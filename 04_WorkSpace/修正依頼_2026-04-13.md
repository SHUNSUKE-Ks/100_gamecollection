## 2. Collection > ストーリー（操作変更）

### 2-1. TextInput 内での改行

- `Ctrl + Enter` で新しいラインを作成し、そのまま次の行を入力できるようにする

### 2-2. キャラクター名入力フロー

- **Character 1** で名前を入力し `TAB` を押したとき：※修正
  - Character 2 が空白であれば → Character 2 の入力へフォーカス移動
  - Character 3 が空白であれば → Character 3 の入力へフォーカス移動
  - Character 4 まで同様に連鎖

### 追加：キャラクター名前入力Enterで会話blockへ移動

### 2-3. キャスト：`Alt + 0` = ナレーション／地の文

- `Alt + 0` はナレーションや地の文に使用するため、**名前は未入力でよい**（空白を許容する）

### 2-4. コメント機能（`Alt + ^`）

- 会話ログには表示されない「コメント（メモ）」を書ける
- **サブヘッダーにコメント一覧を表示**
  - 記入されたコメントの場所へジャンプするボタンを設置

### 2-5. ESC キー

- 現在フォーカスしている input からフォーカスを外す

### 2-6. 会話ブロックの順番入れ替え UI 変更

- **廃止：** 上下移動ボタン
- **新設：** 「二本線」アイコンのドラッグハンドルボタン
  - 押し続けている間（ドラッグ中）は順番を入れ替えられる
  - ドラッグ中、移動先に **青い線** を表示し、挿入位置をわかりやすく示す

## 3. Android 対応

注意：androidの縦画面で会話blockが収まるようにして

### 3-1. サイドバーの開閉式化

- 対象：キャラクター図鑑・プロット手帳のサイドバー
- **問題：** Android ではメイン画面の表示幅が確保されず見にくい
- **提案：** Android 表示時のみ、サイドバーをメインコンテンツの上に **オーバーレイ表示** する（サイドバーがメインにかぶさる形）
