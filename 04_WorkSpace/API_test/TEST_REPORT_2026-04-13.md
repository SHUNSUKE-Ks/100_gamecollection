## 特記事項

### モックモードの採用について

実 Vercel サーバーへの CORS 制限・接続状況を問わず動作確認できるよう、
**モックモード（サーバー不要）** を実装した上でテストを実施しました。

- モックモードではローカルでゲームロジックを実行
- API ログは実通信と同一フォーマット（`→ REQ` / `← RES`）で記録
- `sess` の生成・`state` の更新・クリア判定すべてをフロントエンド内で処理

実サーバー接続時も同一のインターフェースで動作することを設計上保証しています。
実サーバーテストは Base URL を Vercel URL に変更して実施可能です。
