## 新規追加仕様まとめ

### 1. [+] クイックノートボタン（どの画面でも即起動）
- AndroidLayout ヘッダー右上 [+] → モーダルで新規ノート入力
- タイトル・ジャンルを入力 → NovelLibrary に追加（localStorage保存）
- 作成後はライブラリタブに自動遷移

### 2. 左サイドパネル Library（ハンバーガー内）
現状ハンバーガー 6項目 → Library セクションを折りたたみ追加

```
≡ メニュー
├── タイトル一覧（Gallery）
├── ノベルライブラリ
├── プロット手帳
├── レポート
├── 発注書
├── スキーマー確認
└── 📚 Library ▼（折りたたみ）
      ├── TitleDB（作品管理）   ★NEW
      ├── 地名辞典
      ├── キャラクター図鑑
      ├── NPC図鑑
      ├── エネミー図鑑
      ├── アイテム図鑑
      ├── イベントDB
      ├── CG・ギャラリー
      ├── サウンド図鑑
      └── タグDB
```

### 3. TitleDB Relation（全素材に titleId を装着）
全DBに `titleId: string` を追加 → フィルターUI で作品別絞り込み

### 4. 画像 Import（DB別の仕様）
各 DB の詳細項目ページに Import ボタンを追加。

### 5. サウンド アルバム機能
BGM/SE にアルバムを追加。Gallery はアルバム単位でグルーピング表示。

| 用途 | 推奨サイズ | 形式 | 備考 |
|------|-----------|------|------|

## 判断が必要な点

| 質問 | 選択肢 |
|------|--------|
| JSON スキーマ変更後、既存 Firestore データは？ | ① JSON だけ変える ② Firestore も移行 |
| 画像ファイルの保管先 | ① public/ フォルダ（静的） ② Firebase Storage ③ Base64 で JSON 内 |
| 表情差分のグローバル管理 | 設計通り `sharedExpressions` でよいか？ |
| アルバム: BGM と SE を統合 or 分離？ | ① 統合（1つのアルバムに BGM+SE 混在可） ② 分離 |
