# NanoNovel Asset Manager

ブラウザベースで動作する軽量なアセット管理ツールです。
JSONデータの編集と、納品ファイル（`_delivery`）の自動リネーム・フォルダ振分けをGUIで行えます。

## 必要要件
- Node.js (v14以上推奨)

## 起動方法
プロジェクトルート（`c:\GameProjectMaster\kaihatuchuu\NanoNovel`）で以下のコマンドを実行してください。

```bash
node 00_WorkSpace/asset-manager/server.js
```

サーバー起動後、ブラウザで **[http://localhost:3000](http://localhost:3000)** にアクセスします。

## 画面構成と使い方

### 1. JSONファイルの選択 (Sidebar)
左側のサイドバーから編集したいJSONファイル（`characters.json` など）を選択します。選択するとメインエリアにリストが表示されます。

### 2. データの編集 (Main Content)
- **ID**: 識別子（読み取り専用推奨）
- **Name**: 表示名。テキストボックスで編集可能です。
- **Save Folder**: アセットの保存先サブフォルダを選択します。
    - 例: `chara` を選ぶと `src/assets/chara` に保存されます。
    - 階層構造もプルダウンで選択可能です。

### 3. アセットの納品 (Import)
1.  **ファイルの準備**: 納品したい素材（画像など）をデスクトップなどから画面上の任意の行へ**ドラッグ＆ドロップ**します。
2.  **確認**: ドロップした行に「🆕 Ready」と表示され、ファイル名が青色で表示されます。
3.  **実行**: 右端の「⚠️ Import」ボタンをクリックします。
    - ファイルが `src/assets/{Save Folder}` に移動・リネームされます。
    - JSONの `image` パスが自動更新されます。
    - **注意**: この操作は取り消せません。

### 4. フォルダを開く (Open Folder)
「Path Status」列の右下にある 📁 アイコンをクリックすると、その行で指定されている保存先フォルダ（エクスプローラー）が開きます。
納品されたファイルを直接確認するのに便利です。

### 5. 保存 (Save Changes)
右上の「💾 Save Changes」ボタンを押すと、現在のJSONの変更内容（名前の変更など）がファイルに書き込まれます。
（Import操作時は自動で保存されます）
