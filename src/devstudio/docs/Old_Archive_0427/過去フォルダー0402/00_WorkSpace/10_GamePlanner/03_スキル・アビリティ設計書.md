# スキル・アビリティ設計書

ゲーム内で使用するスキルとアビリティのカラム定義です。

---

## ⚡ スキル (アクティブスキル) カラム一覧

| カラム名 | 型 | 必須 | 説明 | 例 |
|----------|-----|------|------|-----|
| `id` | string | ✅ | 一意のID | `"skill_fire_001"` |
| `name` | string | ✅ | 表示名 | `"ファイアボール"` |
| `type` | string | ✅ | スキル種別 | `"attack"`, `"heal"`, `"buff"` |
| `icon` | string | - | アイコン画像 | `"skill/fire.png"` |
| `element` | string | - | 属性 | `"fire"`, `"ice"`, `"none"` |
| `mpCost` | number | ✅ | MP消費量 | `10` |
| `tpCost` | number | - | TP消費量 | `0` |
| `cooldown` | number | - | クールダウン (ターン) | `0` |
| `power` | number | ✅ | 威力 (基本値) | `120` |
| `hitCount` | number | - | ヒット回数 | `1` |
| `accuracy` | number | - | 命中率 (%) | `95` |
| `critBonus` | number | - | クリティカル補正 (%) | `0` |
| `target` | string | ✅ | 対象タイプ | `"enemy_single"`, `"enemy_all"`, `"ally_single"` |
| `range` | string | - | 射程 | `"melee"`, `"ranged"` |
| `effects` | object[] | - | 追加効果 | 下記参照 |
| `animation` | string | - | 演出ID | `"anim_fire_001"` |
| `description` | string | - | 説明文 | `"炎の球を放つ。"` |
| `learnLevel` | number | - | 習得レベル | `5` |
| `learnClass` | string[] | - | 習得職業 | `["mage"]` |

---

## 🌟 アビリティ (パッシブスキル) カラム一覧

| カラム名 | 型 | 必須 | 説明 | 例 |
|----------|-----|------|------|-----|
| `id` | string | ✅ | 一意のID | `"ability_hp_up_001"` |
| `name` | string | ✅ | 表示名 | `"HP強化"` |
| `type` | string | ✅ | タイプ | `"stat_boost"`, `"resistance"`, `"special"` |
| `icon` | string | - | アイコン画像 | `"ability/hp_up.png"` |
| `effect` | object | ✅ | 効果内容 | 下記参照 |
| `cost` | number | - | コスト (装備ポイント) | `3` |
| `stackable` | boolean | - | 重複装備可能か | `false` |
| `description` | string | - | 説明文 | `"最大HPが10%上昇する。"` |
| `learnLevel` | number | - | 習得レベル | `10` |
| `learnClass` | string[] | - | 習得職業 | `["warrior", "knight"]` |

---

## 🎯 ターゲットタイプ定義

| ID | ラベル | 説明 |
|----|--------|------|
| `enemy_single` | 敵単体 | 敵1体を選択 |
| `enemy_all` | 敵全体 | 敵全員に効果 |
| `enemy_random` | 敵ランダム | ランダムな敵に効果 |
| `ally_single` | 味方単体 | 味方1体を選択 |
| `ally_all` | 味方全体 | 味方全員に効果 |
| `self` | 自分 | 自分自身のみ |
| `all` | 全体 | 敵味方全員 |

---

## 🔥 属性定義

| ID | ラベル | 相性 |
|----|--------|------|
| `none` | 無属性 | 相性なし |
| `fire` | 炎 | 氷に強い、水に弱い |
| `ice` | 氷 | 風に強い、炎に弱い |
| `wind` | 風 | 土に強い、氷に弱い |
| `earth` | 土 | 雷に強い、風に弱い |
| `thunder` | 雷 | 水に強い、土に弱い |
| `water` | 水 | 炎に強い、雷に弱い |
| `light` | 光 | 闇に強い |
| `dark` | 闇 | 光に強い |

## 🔧 効果 (effects) 定義例

```json
"effects": [
    {
        "type": "damage",
        "element": "fire",
        "power": 120,
        "scaling": "matk"
    },
    {
        "type": "status",
        "status": "burn",
        "chance": 30,
        "duration": 3
    }
]
```

### 効果タイプ一覧

| type | 説明 | 必要パラメータ |
|------|------|----------------|
| `damage` | ダメージ | `power`, `element`, `scaling` |
| `heal` | 回復 | `power`, `scaling` |
| `status` | 状態異常付与 | `status`, `chance`, `duration` |
| `buff` | バフ付与 | `stat`, `value`, `duration` |
| `debuff` | デバフ付与 | `stat`, `value`, `duration` |

```json

    "id": "skill_fire_001",
    "name": "ファイアボール",
    "type": "attack",
    "icon": "skill/fire.png",
    "element": "fire",
    "mpCost": 15,
    "power": 120,
    "accuracy": 95,
    "target": "enemy_single",
    "range": "ranged",
    "effects": [
        {
            "type": "damage",
            "element": "fire",
            "power": 120,
            "scaling": "matk"
        },
        {
            "type": "status",
            "status": "burn",
            "chance": 20,
            "duration": 3
        }
    ],

}
```
