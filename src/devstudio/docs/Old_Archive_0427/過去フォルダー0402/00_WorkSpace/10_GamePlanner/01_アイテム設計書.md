# アイテム設計書

ゲーム内で使用するアイテムのカラム定義です。

---

## 📋 アイテムカラム一覧

| カラム名 | 型 | 必須 | 説明 | 例 |
|----------|-----|------|------|-----|
| `id` | string | ✅ | 一意のID (英数字_) | `"potion_hp_001"` |
| `name` | string | ✅ | 表示名 | `"ポーション"` |
| `category` | string | ✅ | カテゴリID | `"consumable"`, `"equipment"`, `"key"` |
| `icon` | string | - | アイコン画像パス | `"item/potion.png"` |
| `tags` | string[] | - | 検索用タグ | `["HEAL", "HP"]` |
| `rarity` | number | - | レアリティ (1-5) | `1` |
| `price` | number | - | 購入価格 | `50` |
| `sellPrice` | number | - | 売却価格 | `25` |
| `effect` | string | ✅ | 効果説明 | `"HPを50回復する"` |
| `description` | string | - | フレーバーテキスト | `"基本的な回復薬。"` |
| `stackable` | boolean | - | スタック可能か | `true` |
| `maxStack` | number | - | 最大スタック数 | `99` |
| `usableInBattle` | boolean | - | 戦闘中使用可 | `true` |
| `usableInField` | boolean | - | フィールド使用可 | `true` |

---

## 🏷️ カテゴリ定義

| ID | ラベル | 説明 |
|----|--------|------|
| `consumable` | 消費アイテム | 使用すると消える (回復薬、バフアイテム) |
| `equipment` | 装備品 | 装備して効果発揮 (武器・防具) |
| `key` | 貴重品 | イベントアイテム、売却不可 |
| `material` | 素材 | クラフト・強化用素材 |
| `treasure` | お宝 | 売却用アイテム |

---

## 🔖 タグ例

| タグ | 用途 |
|------|------|
| `HEAL` | 回復系 |
| `BUFF` | 一時強化 |
| `DEBUFF` | 敵弱体化 |
| `DAMAGE` | ダメージアイテム |
| `REVIVE` | 蘇生 |
| `CURE` | 状態異常回復 |
| `WEAPON` | 武器 |
| `ARMOR` | 防具 |
| `ACCESSORY` | アクセサリ |

---

```json
    "sellPrice": 25,

    "description": "薬草を煮詰めて作った基本的な回復薬。",
    "stackable": true,
    "maxStack": 99,
    "usableInBattle": true,

}
```
