# 🎵 SoundCreator (サウンドクリエイター) フォルダ

BGM・効果音の制作・納品に関する資料をまとめたフォルダです。

---

## 📁 フォルダ構成

```
50_SoundCreator/
├── README.md                  (このファイル)
├── 01_音声規格書.md            (形式・ビットレートルール)
├── 02_命名規則.md              (ファイル名ルール)
├── 発注リスト/                 (制作依頼一覧)
└── 納品/                      (完成素材配置)
```

---

## ✅ 作業開始前に確認すること

| 確認項目 | 取得先 | 状態 |
|---------|-------|------|
| 世界観設定 | GamePlanner | [ ] |
| シーンリスト・雰囲気 | ScenarioWriter | [ ] |

| 命名規則 | 本フォルダ `02_命名規則.md` | [ ] |

---

## 🎼 音声規格 (クイックリファレンス)

| 素材タイプ | 形式 | サンプルレート | ビットレート |
|-----------|------|---------------|-------------|
| BGM | MP3 / OGG | 44100 Hz | 192 kbps |
| SE | MP3 / OGG | 44100 Hz | 128 kbps |
| ボイス | MP3 / OGG | 44100 Hz | 128 kbps |

---

## 📝 命名規則 (クイックリファレンス)

```
{カテゴリ}_{名前}.{拡張子}
```

### 例
| 素材 | ファイル名 |
|------|-----------|
| バトルBGM | `bgm_battle_01.mp3` |
| フィールドBGM | `bgm_field_forest.mp3` |
| 剣の効果音 | `se_sword_slash.mp3` |
| 決定音 | `se_ui_decide.mp3` |

---

## 🏷️ カテゴリプレフィックス

| プレフィックス | カテゴリ | 用途 |
|---------------|---------|------|
| `bgm_` | BGM | ループ再生用 |
| `se_` | 効果音 | 単発再生用 |
| `me_` | ME (ジングル) | レベルアップ等 |
| `voice_` | ボイス | キャラボイス |

---

## 🔗 他部署との連携

### 必要な情報 (受け取る)

| 情報 | 提供元 | 用途 |
|------|-------|------|
| 世界観・雰囲気 | GamePlanner | BGMの方向性 |
| シーンリスト | ScenarioWriter | 場面に合わせた曲制作 |
| 必要SE一覧 | GamePlanner | SE制作リスト |

### 提供する情報 (渡す)

| 情報 | 提供先 | 用途 |
|------|-------|------|
| 完成素材 | Programmer | ゲームへの組み込み |
| BGM/SEタグ一覧 | ScenarioWriter | シナリオでの指定用 |

---

## 📦 納品方法

1. 完成した素材を `納品/` フォルダに配置
2. または直接 `src/assets/bgm/` `src/assets/se/` へ配置
