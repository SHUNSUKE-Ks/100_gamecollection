# 命名規則

音声素材のファイル名ルールです。

---

## 📝 基本フォーマット

```
{カテゴリ}_{名前}[_{番号}].{拡張子}
```

**ルール:**
- 全て **半角英数字** + アンダースコア
- **小文字のみ** 使用
- スペース、日本語禁止

---

## 🏷️ カテゴリプレフィックス

| プレフィックス | カテゴリ | 例 |
|---------------|---------|-----|
| `bgm_` | BGM | `bgm_battle_01.mp3` |
| `se_` | 効果音 | `se_sword_slash.mp3` |
| `me_` | ME (ジングル) | `me_levelup.mp3` |
| `voice_` | ボイス | `voice_hero_attack.mp3` |

---

## 🎼 BGM命名例

```
bgm_{シーン/用途}[_{番号}].mp3
```

| シーン | ファイル名 |
|-------|-----------|
| タイトル | `bgm_title.mp3` |
| フィールド | `bgm_field_01.mp3` |
| 森 | `bgm_field_forest.mp3` |
| 街 | `bgm_town.mp3` |
| バトル通常 | `bgm_battle_01.mp3` |
| ボス戦 | `bgm_battle_boss.mp3` |
| イベント | `bgm_event_sad.mp3` |

---

## 🔊 SE命名例

```
se_{カテゴリ}_{アクション}.mp3
```

### UI系

| 内容 | ファイル名 |
|------|-----------|
| 決定 | `se_ui_decide.mp3` |
| キャンセル | `se_ui_cancel.mp3` |
| カーソル移動 | `se_ui_cursor.mp3` |
| エラー | `se_ui_error.mp3` |

### 戦闘系

| 内容 | ファイル名 |
|------|-----------|
| 剣攻撃 | `se_sword_slash.mp3` |
| 魔法発動 | `se_magic_fire.mp3` |
| ダメージ | `se_damage_01.mp3` |
| 回復 | `se_heal.mp3` |
| 勝利 | `se_battle_win.mp3` |

### 環境系

| 内容 | ファイル名 |
|------|-----------|
| ドアを開ける | `se_door_open.mp3` |
| 足音 | `se_footstep.mp3` |
| 雨音 | `se_rain.mp3` |

---

## 🎉 ME (ジングル) 命名例

```
me_{イベント}.mp3
```

| イベント | ファイル名 |
|---------|-----------|
| レベルアップ | `me_levelup.mp3` |
| アイテム入手 | `me_item_get.mp3` |
| クエスト完了 | `me_quest_clear.mp3` |
| ゲームオーバー | `me_gameover.mp3` |

---

## ⚠️ 禁止事項

- ❌ 日本語: `bgm_バトル.mp3`
- ❌ スペース: `se sword slash.mp3`
- ❌ 大文字混在: `BGM_Battle.mp3`
- ❌ ハイフン: `se-sword-slash.mp3`
