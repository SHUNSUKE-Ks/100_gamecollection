# キャラクター設定資料: Novel01_SampleGame

## 1. アルト (Alto)
**役割:** 主人公 / 冒険者
**年齢:** 19歳
**性別:** 男性
**一人称:** 俺
**二人称:** お前、あんた、(名前)
**性格:** 
少し無鉄砲だが、困っている人を放っておけない正義感を持つ。
「塔」の頂上にあるという「真実」を求めているが、その動機は亡き父の遺言にある。
**外見イメージ:**
- 軽装の鎧、片手剣。
- 赤髪、活発な印象。
**タグ:** `chara_alto`
**表情差分:**
- `normal`: 真剣な顔
- `smile`: 屈託のない笑顔
- `angry`: 怒り
- `sweat`: 困り顔

## 2. エリス (Eris)
**役割:** メインヒロイン / 謎の少女
**年齢:** 外見16歳前後（実年齢不明）
**性別:** 女性
**一人称:** 私（わたし）
**二人称:** あなた、(名前)
**性格:** 
記憶喪失だが、知識だけはあるため時折大人びた発言をする。
感情表現は少し乏しいが、アルトとの冒険を通じて感情を取り戻していく。
**外見イメージ:**
- 白を基調としたローブ。
- 銀髪、蒼い瞳。どこか儚げな雰囲気。
- 首元に不思議な石のペンダント。
**タグ:** `chara_eris`
**表情差分:**
- `normal`: 無表情・静か
- `smile`: 控えめな微笑み
- `worry`: 伏し目がち・不安
- `surprise`: 目を見開く

## 3. ギルドマスター・ガイン (Gane)
**役割:** チュートリアル / サブキャラクター
**年齢:** 45歳
**性別:** 男性
**一人称:** ワシ
**二人称:** 坊主、お前さん
**性格:** 
豪快で声が大きい。塔に挑む若者たちを厳しくも温かく見守る。
元は凄腕の冒険者だったらしい。
**外見イメージ:**
- 筋肉質、髭面。
- ギルドの制服（エプロン？）姿。
**タグ:** `chara_gane`
**表情差分:**
- `normal`: 腕組み
- `laugh`: 大笑い
- `angry`: 説教
