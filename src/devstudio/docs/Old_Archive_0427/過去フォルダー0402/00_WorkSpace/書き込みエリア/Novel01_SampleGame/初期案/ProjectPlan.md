# 企画書: Novel01_SampleGame

## 1. ゲームコンセプト (Phase 1)
**タイトル:** 「幻想の塔と忘却の少女」 (仮)
**ジャンル:** 短編ファンタジーノベル
**ターゲット層:** 
- 王道ファンタジーが好きな層
- 短時間で物語を楽しみたい層
**コンセプト/フック:** 
- 「塔」を登るごとに明かされる世界の謎と少女の記憶。
- シンプルな選択肢で結末が変わるマルチエンディング形式。

## 2. 世界観設定
**舞台:** 
水晶の塔「アイオーン」。雲を突き抜けるほど高く、最上階には「願いを叶える秘宝」があると言われている。
**重要要素:**
- **魔力結晶:** 世界のエネルギー源。塔の動力源でもある。
- **記憶の欠片:** 塔の各階層に散らばる、過去の冒険者たちの記憶。

## 3. キャラクター設定
### 主人公 (プレイヤー)
**名前:** アルト (変更可)
**役割:** 新米冒険者
**設定:** 
村の伝説を確かめるために塔に挑む若者。剣の腕は未熟だが、強い意志と「視る力（隠されたものを見抜く才能）」を持つ。

### メインヒロイン
**名前:** エリス
**役割:** 塔の案内人 / 謎の少女
**設定:** 
塔の入り口で主人公を待ち受けていた少女。自分の名前以外の記憶を失っている。
塔の機能や魔物の知識に詳しいが、なぜ知っているのか自分でも分からない。

### サブキャラクター
**名前:** ギルドマスター・ガイン
**役割:** チュートリアル役
**設定:** 
塔の麓にある冒険者ギルドの親父。豪快だが面倒見が良い。

## 4. 基本システム
**ゲームサイクル:** 
会話パート(Novel) ➝ 探索・選択(Choice) ➝ 運命分岐(Branch)

**ユニーク要素:**
- **「追憶」システム:** 特定のアイテムを入手することで、エリスの過去の記憶（サブストーリー）が解放される。
