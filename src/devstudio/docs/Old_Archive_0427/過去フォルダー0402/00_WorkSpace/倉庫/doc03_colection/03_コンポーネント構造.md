# コンポーネント構造

## 依存関係図

```
CollectionScreen
├── useViewMode (Hook)
├── useGameStore (Zustand)
├── ViewSwitcher
├── TableView ←──┐
├── GalleryView ←┼─ data-views
├── KanbanView ←─┘
├── CharacterDetailView ←──┐
├── NPCDetailView ←────────┤
├── EnemyDetailView ←──────┼─ parts/collection/specific
├── BackgroundDetailView ←─┤
├── BGMPlayerView ←────────┘ (uses useBGMPlayer)
├── StoryView ←─ parts/collection/story
└── StudioScreen ←─ screens/12_Studio
```

## 主要Props

### TableView
```typescript
interface TableViewProps {
  data: any[];
  columns: TableColumn[];
  onEdit?: (item: any) => void;
  onDelete?: (item: any) => void;
}

interface TableColumn {
  key: string;
  label: string;
  render?: (value: any, item: any) => ReactNode;
}
```

### ViewSwitcher
```typescript
interface ViewSwitcherProps {
  currentView: 'list' | 'gallery' | 'kanban' | 'custom';
  onViewChange: (view: CollectionViewType) => void;
}
```

### 詳細View共通
```typescript
interface DetailViewProps {
  data: DataType[];  // 各データ型の配列
}
```

## タブ構成

### PrimaryTab (第1階層)
| ID | 表示名 | 動作 |
|----|-------|------|
| item | アイテム | 準備中 |
| equipment | 装備 | 準備中 |
| skill | スキル | 準備中 |
| ability | 能力 | 準備中 |
| **story** | ストーリー | StoryView表示 |
| **library** | ライブラリー | SecondaryTab表示 |
| **sound** | 音 | BGMPlayerView表示 |
| keymap | キーマップ | 準備中 |
| **studio** | 工房 | StudioScreen表示 |

### SecondaryTab (第2階層 - library選択時)
| ID | 表示名 | データソース | View |
|----|-------|-------------|------|
| place | 地名辞典 | backgrounds.json | BackgroundDetailView |
| character | キャラクター図鑑 | characters.json | CharacterDetailView |
| npc | NPC図鑑 | npcs.json | NPCDetailView |
| enemy | エネミー図鑑 | enemies.json | EnemyDetailView |
| item_dict | アイテム図鑑 | items.json | TableView |
| event | イベントDB | events.json | TableView |
| cg | CG・ギャラリー | gallery.json | GalleryView |
| sound_db | サウンド図鑑 | bgm+se.json | BGMPlayerView |
