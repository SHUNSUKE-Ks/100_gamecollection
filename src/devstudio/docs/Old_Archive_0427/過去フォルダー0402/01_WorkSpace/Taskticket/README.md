# Task Ticket System (AI用指示書)

このフォルダには、ゲーム開発の各フェーズにおける具体的な作業指示を記した「タスクチケット」が格納されています。

## 使い方
AI（ChatGPT, Claude, Geminiなど）を作業指示に使う際、以下の手順で進めてください。

1.  現在の開発状況に合わせたフェーズのチケットを選択する。
2.  そのチケットの内容をAIに渡し、「この指示に従って作業を開始してください」と伝える。
3.  AIが必要なファイルや情報を求めてきたら、順次提供する。
