## 実装した機能

### 1. 発注書インボックス（DocumentInboxView）

**目的**: Collection / Report 画面に Notion スタイルのドキュメント管理を追加し、シナリオライター向けの発注書フォーマット・ルール・納品スキーマを管理できるようにする。

**作成ファイル**:
- `src/core/types/document.ts` — 型定義（OrderDocument, OrderBrief, DeliverySchema, etc.）
- `src/data/collection/documents.json` — サンプルドキュメント5件、執筆ルール11件
- `src/parts/collection/document/DocumentInboxView.tsx` — メインコンポーネント

**主な機能**:
- 3つのサブタブ：インボックス（テーブル一覧）、執筆ルール、スキーマ一覧
- フィルター：検索・ステータス・カテゴリ・ロール（optgroup ドロップダウン）・ソート
- 右ドロワー：発注内容 / 納品スキーマ / バリデーション の3タブ
- **ロールカラム**：Scenario / Designer / Sound グループバッジ + サブロールラベル

**修正ファイル**:
- `src/screens/11_Collection/CollectionScreen.tsx` — `'document'` タブ追加
- `src/parts/collection/report/ReportView.tsx` — `'document'` タブ追加

---

### 2. スキーマーショート（SchemaShortView）

**目的**: Story サブタブに AI 向けの最小 JSON スキーマを表示・コピーできる画面を追加。シナリオ担当 AI がスキーマに従って JSON を書けば、ノベルゲームが完成する。

**作成ファイル**:
- `src/parts/collection/story/SchemaShortView.tsx`

**スキーマバリアント**:

| バージョン | 内容 |
|-----------|------|
| **v1.1** | 基本（タイトル・エピソード・会話） |
| **v1.2** | CHOICE分岐・returnTo・line.id を追加 |
| **v2.1** | 背景・キャラクター表示を追加 |
| **v3.1** | Bool フラグ管理 + 加算軸パラメーター（勇気・好感度等）を追加 |

**機能**:
- Ver1 は `▽` ドロップダウン（v1.2 が最新としてデフォルト表示）
- 各バージョンバナー右端に **AllCopy** ボタン
- フィールドごとの **CopyButton**（コピー完了フィードバック付き）
- AI プロンプトテンプレートをバージョンごとに生成・コピー
- `initialVersion` prop で外部からバージョン指定可能（DeepLink 対応）

---

### 3. ノベルライブラリ（NovelLibraryView + NovelDetailView）

**目的**: Story サブタブにインポートした JSON ノベルを Notion 風リスト/ギャラリーで管理し、会話ログビューで閲覧できるようにする。

**作成ファイル**:
- `src/parts/collection/story/NovelLibraryView.tsx`
- `src/parts/collection/story/NovelDetailView.tsx`

**NovelLibraryView 機能**:
- リストビュー：Title + バージョンバッジ + 行数、Dict、Genre、作成日、進捗バー
- ギャラリービュー：カード形式（サムネ = 最初のセリフテキスト）+ 進捗バー
- **インポートモーダル（2ステップ）**：① JSON 貼り付け → ② タイトル・Dict・Genre 編集
- **進捗計算（DONE ルール）**：DONE01 = 全 line に text が存在する
- localStorage（`novel_library_v1`）で永続化

**NovelDetailView 機能**:
- ドラッグ＆ドロップで Scene 並び替え可能なサイドバーステッパー
  - Episode 行（開閉式）→ Scene 行（GripVertical アイコン）
  - CHOICE シーン：🔀 アイコン、returnTo：↪ アイコン
- メインエリアに **会話ログ**：
  - ナレーション（左ボーダー＋イタリック）
  - セリフ（キャラクターアイコン＋スピーカー名＋テキストバブル）
  - **ChoiceCard**：選択肢テキスト・遷移先・結果・エフェクトバッジ
  - **EffectsBadge**：フラグバッジ（紫）・軸バッジ（黄）

**修正ファイル**:
- `src/parts/collection/story/StoryView.tsx` — `schema` / `library` サブタブ追加、詳細ページ遷移ロジック

## 対処した技術的問題

|------|------|
| Tailwind v4 クラス名変更 | `flex-shrink-0` → `shrink-0`、`w-[480px]` → `w-120`、`min-h-[200px]` → `min-h-50` |
