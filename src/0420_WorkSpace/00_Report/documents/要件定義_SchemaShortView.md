# 要件定義 — SchemaShortView

> 作成日: 2026-04-20  
> 対象ファイル: `src/parts/collection/story/SchemaShortView.tsx`  
> 配置場所: Collection → Story → 📄 スキーマーショート タブ

---

## 概要

ノベルエンジン用 JSON スキーマの「クイックリファレンス」画面。  
AIにシナリオを渡すとき・自分でJSONを書くときの**仕様確認・プロンプトコピー・サンプル参照**を1画面で完結させる。

---

## バージョン体系

| バージョンID | ラベル | 概要 |
|---|---|---|
| `v11` | Ver 1.1 | テキスト・話者のみ。最小構成 |
| `v12` | Ver 1.2 | CHOICE・分岐・合流追加 |
| `v21` | Ver 2.1 | 背景ID・キャラクターID・表情差分追加 |
| `v31` | Ver 3.1 | Bool フラグ・数値軸パラメーター追加（conditions/effects） |
| `v41` | Ver 4.1 | 4ファイル分離構成（events / chats / choices / state） |

### バージョングループルール
- **Ver 1.x** はプルダウン（▽ドロップダウン）でまとめる。最新は v1.2
- **Ver 2.1 以降**は個別ボタンで横並び表示
- 選択中バージョンは黄色ハイライト（`bg-yellow-500/20 text-yellow-300`）

## 各セクション詳細

### 1. ツールバー（固定ヘッダー）

| 要素 | 動作 |
|---|---|
| Ver 1.x ドロップダウン | v1.1 / v1.2 を切替。外クリックで閉じる |
| Ver 2.1〜4.1 ボタン | クリックで即時バージョン切替 |
| Import ボタン | JSONインポートモーダルを開く |
| Export ボタン | インポート済みJSON or スキーマサンプルをファイルDL |
| Ref ボタン | 右側リファレンスパネルをトグル |
| インポート済みバッジ | インポート中のみ表示。×で解除 |

### 2. バージョンバナー

- バージョンごとにアクセントカラーを変更（border / icon / title color）

| バージョン | カラー |
|---|---|
| v1.1 | slate（グレー） |
| v1.2 | indigo（インディゴ） |
| v2.1 | pink（ピンク） |
| v3.1 | yellow（イエロー） |
| v4.1 | emerald（グリーン） |

- 右端に `Copy All`（全JSONコピー）ボタン常設

### 3. SpecTable（フィールド仕様テーブル）

- **トグル式**：ヘッダークリックで展開 / 折りたたみ
- 基本フィールド（全バージョン共通）+ 拡張フィールド（差分のみ）の2段構成
- v4.1 は基本フィールドのみ（SPEC_V41 が独自定義）
- 各行：`フィールド名（mono/黄）/ 型（cyan）/ 必須 or 任意 / 説明`

### 4. スキーマサンプル

- **トグル式**：クリックで `<pre>` を展開
- 右端に Copy ボタン（クリックで JSON 全文コピー、1.6s 後リセット）

### 5. インポートデータプレビュー（条件付き表示）

- インポート済みの場合のみ表示
- バージョン・タイトル・エピソード数・会話行数をカード4枚で要約
- 全JSONを `<pre>` で表示

### 6. AI 指示テンプレート

- **トグル式**：クリックでプロンプト `<pre>` を展開
- 右端に Copy ボタン
- バージョンごとに追加テキストあり：
  - v1.2 / v2.1：CHOICEの遷移・returnToの注意書き
  - v3.1：axesの自由命名に関する注意書き
  - v4.1：4ファイル分離構成の出力指示

### 7. リファレンスパネル（右サイドバー、幅 w-72）

- `Ref` ボタンで右サイドから出現 / 消える
- 内容：
  - キャラクターID一覧（クリックでクリップボードコピー）
  - characterEmotion 値一覧（normal / smile / angry / sad / surprised / null）
  - 背景ID一覧（BG_MAGIC_TOWER 等、クリックでコピー）
  - フラグ名の例（visited_tower 等）
  - 軸パラメーター例（courage / affection 等）

---

## インポートモーダル

- フルスクリーンオーバーレイ（backdrop-blur）
- テキストエリアに JSON を貼り付けて「インポート」
- JSON パース失敗時はエラーメッセージ表示
- 成功時：`importedJson` state に保存、プレビューセクションに反映
- Export ボタンはインポートデータを優先してダウンロード

---

## CopyButton の仕様

- `Copy` → クリックでクリップボードへ書き込み
- 書き込み成功後 1.6 秒間 `Copied!` + チェックアイコンに変化
- `e.stopPropagation()` で親要素のクリックイベントを伝播させない

