## このツールの目的

ゲーム内の「会話シーン」の台本を書くためのメモツール。

- **PlotCard** 1枚 = シーン1つ（例：「主人公が村人に話しかける」）
- カード内に **PlotLine（セリフ行）** を複数並べて会話の流れを作る
- 完成したカードは後でノベルゲームの JSON スキーマに変換される

MEMO02＿PlotCard plotCardはScenarioのたたき台の下が気で使う用語？（箇条書きや列挙したもの）実際の会話のファイルははなんと呼ぼうか？

製作の流れ的に、簡単にイベントカードや会話Cardを書いていって、あとで物語に埋め込んでいけるようにしたい。

埋め込んだ後はステッパー形式のフローチャートで途中に会話用のブロックを挟んでいけるようにする。クリックすると階層を下げてメインビューに会話が映し出される。

## データ型定義

### PlotStatus

```ts
type PlotStatus = "idea" | "draft" | "fixed";
// idea  = アイディア（紫）
// draft = 下書き（黄）
// fixed = 確定（緑）
```

### PlotLine（1行）

```ts
interface PlotLine {
  id: string; // uid
  speaker: string; // キャラ名。空文字 = ナレーション
  text: string; // 最大60文字
  isComment?: boolean; // true = コメント行（JSON出力されない）
}
```

### PlotCard（1シーン）

```ts
interface PlotCard {
  id: string;
  title: string;
  status: PlotStatus;
  episodeId: string; // episodes.json の id
  chapterId: string; // episode.chapters[].id
  sceneTag: string; // 自由記述
  castSlots: [string | null, string | null, string | null, string | null]; // Alt+1〜4 用
  lines: PlotLine[]; // 1件以上必須
  updatedAt?: number; // UNIX ms
}
```

### 行の種別ルール

| 種別         | speaker        | 見た目                         | JSONへの出力 |
| ------------ | -------------- | ------------------------------ | ------------ |
| セリフ       | キャラクター名 | 通常                           | される       |
| ナレーション | `""` 空文字    | 薄いボーダー左付き、斜体       | される       |
| コメント     | 任意           | グレー背景 `//` プレフィックス | **されない** |

## コンポーネント Props 定義

`PlotNotebookShell` が受け取る Props（`schema_plot.json` v2.0 準拠）:

```ts
interface PlotNotebookProps {
  // データ
  cards: PlotCard[];
  characters: Character[]; // キャストスロット選択用
  episodes: Episode[]; // Episode / Chapter 選択用

  // 状態
  isLoading: boolean;
  error: string | null;

  // コールバック
  onAddCard: () => void;
  onDeleteCard: (id: string) => void;
  onUpdateCard: (id: string, patch: Partial<PlotCard>) => void;
  onAddLine: (cardId: string, afterIdx: number) => void;
  onDeleteLine: (cardId: string, lineId: string) => void;
  onUpdateLine: (cardId: string, lineId: string, patch: Partial<PlotLine>) => void;
  onReorderLines: (cardId: string, newLines: PlotLine[]) => void;

  // オプション
  isAIEnabled?: boolean;
  onAIGenerate?: (prompt: string) => Promise<PlotCard[]>;
  initialCardId?: string | null;
}
```

## キーボードショートカット仕様

| キー                 | 動作                                       | 条件                                  |
| -------------------- | ------------------------------------------ | ------------------------------------- |
| `Ctrl+Enter`         | フォーカス行の下に新しい行を追加           | textarea フォーカス中                 |
| `Alt+1〜4`           | castSlots[n-1] のキャラ名を speaker に入力 | textarea / speaker input フォーカス中 |
| `Alt+0`              | speaker を空（ナレーション）にする         | 同上                                  |
| `Alt+A`              | サイドバー開閉トグル                       | 常時                                  |
| `ESC`                | フォーカスを外す                           | 常時                                  |
| `Backspace` on empty | 行を削除して前の行にフォーカス             | textarea が空でフォーカス中           |

## 画面フェーズ定義

| Phase           | 説明               | 遷移                         |
| --------------- | ------------------ | ---------------------------- |
| `LOADING`       | 初回ロード中       | 完了→ EMPTY or CARD_SELECTED |
| `ERROR`         | ロードエラー       | リトライ→ LOADING            |
| `EMPTY`         | カードが0件        | カード追加→ CARD_SELECTED    |
| `CARD_LIST`     | カードあり・未選択 | クリック→ CARD_SELECTED      |
| `CARD_SELECTED` | カード選択中・待機 | 入力→ EDITING                |
| `EDITING`       | テキスト入力中     | ESC / blur→ CARD_SELECTED    |

## AutoResizeTextarea 仕様

- `rows={1}` 初期値
- テキスト変更のたびに `scrollHeight` を再計算して `height` を設定
- 最大 `13.6rem`（= 8行分）を超えたらスクロール
- コメント行は `fontStyle: italic` / `color: #6b7280`

## キャストスロット UX 仕様

- 空スロット: クリック → ポップアップ（直接入力 input + キャラ一覧リスト）
- 埋まりスロット: 番号 + 名前 + ×クリアボタン
- ワンクリックで選択 or 入力（旧実装の 3 ステップを廃止）
