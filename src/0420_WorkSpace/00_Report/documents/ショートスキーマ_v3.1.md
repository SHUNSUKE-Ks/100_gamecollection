## 1. v3.1 で変わること

<details>
<summary>詳細（クリックで開閉）</summary>

| 変更点 | v2.x | v3.1 |
| --- | --- | --- |
| イベントと会話の関係 | PlotCard に全部まとまっている | **EventCard**（箱）に **SceneCard**（会話）を紐付ける |
| ストーリー全体図 | カードのリスト形式 | **ステッパーフローチャート**で順番が見える |
| 会話へのアクセス | リストからカードを選ぶ | フローチャートのイベントノードをクリック → 会話が展開 |
| ID | 通し番号 | イベントごとに `00` からリセット |

**なぜ変えるか**  
将来、「村の入口に入ったとき自動で会話が始まる」「アイテムを拾ったら会話が発生する」など、  
**イベントトリガー**で会話を呼び出せるようにするため。  
イベントと会話を分けておくと、後から差し替えや再利用がしやすくなります。

</details>

## 2. 全体の構造

<details>
<summary>詳細（クリックで開閉）</summary>

```
Episode（エピソード）
  └── Chapter（チャプター）
        └── EventCard（イベントノード）× 複数  ← フローチャートで並ぶ
              └── SceneCard（会話シーン）× 1〜複数
                    └── ConversationLine（会話行）× 複数
```

**フローチャートのイメージ**

```
[EVENT: 屋根裏に入る] ──→ [EVENT: レミと話す] ──→ [EVENT: 選択肢] ──→ ...
                                  │
                                  ↓ クリックで展開
                          ┌─────────────────┐
                          │  SCENE: 出会い  │
                          │  line-00: ナレーション      │
                          │  line-01: レミのセリフ      │
                          │  line-02: 主人公のセリフ    │
                          └─────────────────┘
```

</details>

## 3. EventCard — イベントノード

<details>
<summary>フィールド一覧（クリックで開閉）</summary>

フローチャート上の「箱」1つ分です。

| フィールド | 型 | 必須 | 説明 |
| --- | --- | --- | --- |
| `id` | 文字列 | ✅ | `evt_` で始まる自動ID（例: `evt_01`） |
| `title` | 文字列 | ✅ | イベントの名前（例：「屋根裏に入る」） |
| `triggerType` | 文字列 | ✅ | トリガーの種類（後述） |
| `triggerCondition` | 文字列 | — | トリガーの条件メモ（自由記述） |
| `episodeId` | 文字列 | ✅ | 属するエピソードID |
| `chapterId` | 文字列 | ✅ | 属するチャプターID |
| `order` | 数値 | ✅ | フローチャート上の順番（0始まり） |
| `sceneIds` | 文字列[] | ✅ | 紐付くSceneCardのIDリスト |
| `status` | SceneStatus | ✅ | 進捗ステータス |
| `updatedAt` | 数値 | — | 最終更新時刻（自動） |

### triggerType の種類

| 値 | 意味 |
| --- | --- |
| `auto` | 前のイベントが終わったら自動で続く |
| `enter_area` | エリアに入ったとき |
| `pick_item` | アイテムを拾ったとき |
| `talk_npc` | NPCに話しかけたとき |
| `battle_end` | バトルが終わったとき |
| `manual` | 手動で呼び出す（テスト・デバッグ用） |

</details>

## 4. SceneCard — 会話シーン

<details>
<summary>フィールド一覧（クリックで開閉）</summary>

EventCard に紐付く「会話シーン」1つ分です。

| フィールド | 型 | 必須 | 説明 |
| --- | --- | --- | --- |
| `id` | 文字列 | ✅ | `sc_` で始まる自動ID（例: `sc_evt01_00`） |
| `eventId` | 文字列 | ✅ | 紐付くEventCardのID |
| `title` | 文字列 | ✅ | シーンのタイトル（例：「レミとの出会い」） |
| `status` | SceneStatus | ✅ | 進捗ステータス |
| `sceneTag` | 文字列 | — | 自由メモ（例：「回想」「選択肢あり」） |
| `castSlots` | [名前 × 4] | ✅ | キーボードショートカット用のキャラ名スロット |
| `lines` | ConversationLine[] | ✅ | 会話行の配列。最低1行必要 |
| `updatedAt` | 数値 | — | 最終更新時刻（自動） |

</details>

## 5. ConversationLine — 会話行

<details>
<summary>フィールド一覧（クリックで開閉）</summary>

v2.x の `PlotLine` に対応します。名前が変わっただけで構造は同じです。

| フィールド | 型 | 必須 | 説明 |
| --- | --- | --- | --- |
| `id` | 文字列 | ✅ | イベントごとに `line-00` からリセット |
| `speaker` | 文字列 | ✅ | 話者名。**空欄 = ナレーション** |
| `text` | 文字列 | ✅ | 台詞・地の文（最大60文字） |
| `isComment` | 真偽値 | — | `true` でゲームに出力されないメモ行になる |

### 行の見た目

| 種類 | speaker | isComment | 見た目 |
| --- | --- | --- | --- |
| セリフ | キャラ名 | false | 通常 |
| ナレーション | **空欄** | false | 左ボーダー・斜体 |
| コメント | 任意 | **true** | グレー背景・`//` プレフィックス |

</details>

## 6. フローチャートビュー（ステッパー）

<details>
<summary>詳細（クリックで開閉）</summary>

```
画面の上部：フローチャート（横スクロール）
┌──────────────────────────────────────────────────────────┐
│  ① 屋根裏に入る → ② レミと話す → ③ 選択肢 → ④ 別れ  │
│      [auto]         [talk_npc]     [auto]      [auto]    │
└──────────────────────────────────────────────────────────┘
                  ↓ ② をクリック
画面の下部：SceneCard エディタ（メインビュー）
┌──────────────────────────────────────────────────────────┐
│  SCENE: 出会い                          [status: draft]  │
│  ────────────────────────────────────────────────────    │
│  キャスト: [1:レミ] [2:主人公] [3:__] [4:__]            │
│  ────────────────────────────────────────────────────    │
│  ⠿  （空欄）    │ 薄暗い屋根裏部屋。埃の匂いが…         │
│  ⠿  レミ        │ ……こんなところに誰かいるの？          │
│  ⠿  主人公      │ あ、ごめん。迷い込んでしまって。      │
│  ────────────────────────────────────────────────────    │
│  [+ 台詞を追加]  [// NOTE]                               │
└──────────────────────────────────────────────────────────┘
```

**操作フロー**

1. フローチャートでイベントをクリック → 下部にSceneCardが展開
2. 台詞を書く
3. 完了したら次のイベントノードへ移動

**イベントの追加**

- フローチャートの末尾にある `[+ イベント追加]` ボタン
- または既存ノードの右クリック → 後ろに挿入

</details>

## 7. IDの採番ルール

<details>
<summary>詳細（クリックで開閉）</summary>

```
EventCard  :  evt_01, evt_02, evt_03 ...  （エピソード内で連番）
SceneCard  :  sc_evt01_00, sc_evt01_01 ...（EventIDごとに 00 からリセット）
ConversationLine:  line-00, line-01 ...   （SceneCardごとに 00 からリセット）
```

**例**

```
evt_02（レミと話す）
  └── sc_evt02_00（出会い）
        ├── line-00  ← このシーンの最初の行
        ├── line-01
        └── line-02

evt_03（選択肢）
  └── sc_evt03_00（別れの選択）
        ├── line-00  ← 別シーンなので再び 00 から
        └── line-01
```

</details>

## 8. ステータス定義

<details>
<summary>詳細（クリックで開閉）</summary>

EventCard と SceneCard で共通のステータスを使います。

| 値 | 表示名 | 色 | 意味 |
| --- | --- | --- | --- |
| `idea` | アイディア | 紫 | まだ構想段階 |
| `draft` | 下書き | 黄 | 書いている途中 |
| `fixed` | 確定 | 緑 | 完成・変換待ち |

</details>

<details>
<summary>詳細（クリックで開閉）</summary>

v2.x と同じです。SceneCard に4スロット。

| スロット | キー | 使い方 |
| --- | --- | --- |

## 10. エピソード・チャプター構造

<details>
<summary>詳細（クリックで開閉）</summary>

v2.x と同じ階層ですが、最下層が PlotCard → EventCard に変わります。

```
Episode（エピソード）
  └── Chapter（チャプター）× 複数
        └── EventCard（イベントノード）× 複数
              └── SceneCard（会話シーン）× 1〜複数
                    └── ConversationLine（会話行）× 複数
```

| 用語 | 例 |
| --- | --- |
| Episode | 「第1話 出会いの日」 |
| Chapter | 「第1章 屋根裏部屋」 |
| EventCard | 「レミと話す」（trigger: talk_npc） |
| SceneCard | 「初めての会話」 |

</details>

## 11. 画面の状態遷移

<details>
<summary>詳細（クリックで開閉）</summary>

| 状態 | 説明 | 次の状態 |
| --- | --- | --- |
| `LOADING` | 起動中・データ読込中 | → FLOWCHART または EMPTY |
| `ERROR` | 読込エラー | → 「Titleへ戻る」ボタン |
| `EMPTY` | イベントが0件 | → FLOWCHART（追加後） |
| `FLOWCHART` | フローチャート表示中・未選択 | → SCENE_EDITING（クリック後） |
| `SCENE_EDITING` | SceneCard 選択・待機 | → EDITING（入力開始） |
| `EDITING` | テキスト入力中 | → SCENE_EDITING（ESC / フォーカス外） |

</details>

## 12. v2.x との用語対照表

<details>
<summary>対照表（クリックで開閉）</summary>

| v2.x の用語 | v3.1 の用語 | 変更の理由 |
| --- | --- | --- |
| `PlotCard` | `EventCard` + `SceneCard` | イベント（箱）と会話（中身）を分離 |
| `PlotLine` | `ConversationLine` | 「会話行」の意味を明確化 |
| カードリスト | フローチャート（ステッパー） | 順番・構造を視覚化 |
| status filter | フローチャート上のバッジ表示 | 全体を見ながら状態確認 |
| episodeId / chapterId | 変わらず | 同じ |
| castSlots | 変わらず | 同じ |

</details>
