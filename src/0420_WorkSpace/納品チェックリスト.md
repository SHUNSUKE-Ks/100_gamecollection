## 納品前の最終確認

| # | チェック | 確認項目 |
|---|---|---|
| 1 | - [ ] | ファイル名はすべて **半角英数字・アンダースコア・ドット**のみか？（スペース・日本語NG） |
| 2 | - [ ] | 画像ファイルは指定のフォルダ（`chara/`, `bg/`, `enemy/`など）に配置されているか？ |
| 3 | - [ ] | 立ち絵・表情差分は **背景透過PNG** になっているか？ |
| 4 | - [ ] | 背景・CGは推奨サイズ **1280×720**（または縦長の場合 720×1280）か？ |
| 5 | - [ ] | アイテムアイコンは推奨サイズ **128×128** か？ |
| 6 | - [ ] | 音楽ファイルの形式は **mp3 または wav** か？ |
| 7 | - [ ] | 記入済みのJSONファイル（`delivery_*.json`）もあわせて納品しているか？ |
